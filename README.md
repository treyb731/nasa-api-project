# NASA API Project

This project uses NASA's Astronomy Picture of the Day API to display a NASA image and description on a webpage.

## How it works

The page sends a request to the NASA API using JavaScript. The API returns data in JSON format, including the image and explanation. The page then displays that information.

## Files

- nasa.html - HTML and JavaScript code
- screenshot.png - shows the project running

## Note

This version uses DEMO_KEY instead of a private API key.
